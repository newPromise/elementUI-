### 关于使用或 （||） 的符号
使用  ||  的符号要和 使用 && 的符号一起来进行使用讲解
在使用这两种操作符的时候，我们要记住一点
>这并非像逻辑运算符，这两个操作符返回的并不是TRUE或者 false ，他们的返回值是两个操作数中的一个（且仅一个） 返回的是两个操作符中的一个
>
使用 || 或者 && 操作符的不同在于，  
使用 || 如果条件判断为 true ,返回第一个操作数的值  
使用 && 如果条件判断为true，返回的是第二个操作数得值  
代码如下：
```
var a = 32;
var b = 42;
console.log(a || b) // 32
console.log(a && b) // 42`
```
